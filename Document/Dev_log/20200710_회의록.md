## 2) 모씨, 어라운드

> 각자 모씨와 어라운드를 사용해보고 느낀 점

- 해시태그를 기반으로 구독 할 수 있는 것이 좋다.
  - 감정 상태가 결정이 되면 게시물마다 감정상태가 있으니까 그 게시물의 특정 감정 상태의 많이 달린 감정이 있을 것.
  - 그 감정을 위주로 달려도 괜찮지 않을까?
- 내용 위주로 피드를 볼 수 있음 
- 거리 보다는 나이! :) 
  - 모씨는 거리가 너무 많이 들어가서 나이가 있으면 좋을 듯
- 모씨는 너무 UI 가 지저분하다. 
  - 너무 많은 태그들이 노출 
- 서비스 리뷰
  - 어라운드: 5.0에서 많이 떨어짐
  - API 중 댓글에 긍/부정을 따진다면 부정적인 글이 표시되면 이 글이 부정적이라 업데이트 될 수 없습니다 라고 만들기
- 어라운드 익명성 보장
  - 현재 플젝에서는 팔로우 기능이 필요한데, 팔로우 기능이 있다면 고민글을 쉽게 올리기가 쉽지 않을 것
- RSS, GCE 포함
- 해시태그나 관심 컨텐츠 볼 수 있는 것이 좋음
- 어라운드 게시물 작성 시 공개가 안됨
- 음성 일기
- 감정 통계
- 투표 기능
- 팔로우 기능 
  - 팔로우 기능이 있어도 다이렉트 메시지를 건네는 것만 없어도 그 사람의 게시물을 받는것도 괜찮다.
  - 개인적으로 연락하는 것은 없는게 좋을 듯 (나만의 공간)
  - 필수가 아니라면 생략하는 것이 개발하기 편함 (다훈님 의견)
